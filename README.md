# 分子可视化工具

> 本项目由 [www.autoa2a.org](https://www.autoa2a.org) 「AI 研究院」**多个 AI 协作自动生成**（共 5 个页面）。在线访问： https://AutoA2A.github.io/autoatoa-mol-viz/

## 网站简介  
本项目是一个基于 Web 的分子可视化工具，使用 Three.js、Mol* 等前端库实现 3D 分子结构的交互式展示与分析。支持 PDB、SDF、MOL2 等常见格式的上传、渲染、剖面、测量及颜色方案切换，旨在为科研人员、教师和学生提供低门槛的可视化平台。  

## 页面与功能  
网站共包含 5 个页面：  
1. **首页（index.html）**：项目概览、快速上手指南与示例分子。  
2. **上传页（upload.html）**：文件拖拽或粘贴 SMILES，实时预览分子。  
3. **查看页（viewer.html）**：3D 查看器，支持旋转、缩放、剖面、距离/角度测量、多种着色模式。  
4. **分析页（analysis.html）**：提供二级结构识别、氢键检测、表面静电势等简单分析功能。  
5. **关于页（about.html）**：技术栈、致谢与联系方式。  

## 多 AI 协作与验收  
项目由三个 AI 协作者完成：AI‑A 负责需求梳理与架构设计，AI‑B 负责前端实现与交互细节，AI‑C 负责代码审查、性能测试与文档撰写。每个阶段均采用结对评审 + 自动化单元测试（Jest）以及手动功能检查清单，确保所有功能在 Chrome、Firefox、Safari 上正常运行。  

## GitHub Pages 部署与访问(入口 index.html)  
将仓库推送至 GitHub 后，在 Settings → Pages 中选择 main 分支 /docs（或根目录）作为源，启用后即可通过 https://<username>.github.io/<repo>/ 访问。入口文件为 index.html，直接打开即可进入首页开始使用分子可视化工具。
